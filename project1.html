The write operation puts data into a specified address in the memory. Data units go into the memory during write operation. • The read operation copies data out of a specified address in the memory. Data units come out of the memory during a read operation. • The data go in and out of the memory on set of lines called data bus. The data bus is bidirectional, which mean that data can go in either direction. • In case of byte-organized memories, data bus has at least eight lines so that all eight bits in a selected address are transferred in parallel. • For a read or write operation, an address is selected by placing a binary code representing the desired address on a set of lines called the address bus. The address bus is unidirectional, which mean that data can go in either direction 


The two major categories of semiconductor memories are the RAM and ROM
• RAM stands for Random Access Memory.
• It is a type of memory in which all addresses are accessible in equal amount of time and
can be selected in any order for a read or write operation.
• All RAMs have both read and write capability.
• Because RAM loses stored data when the power is turned off, they are called volatile
memory.
• ROM stands for Read Only Memory.
• It is where data are stored permanently or semi-permanently.
• Data can be read from a ROM, but there is no write operation as in RAM.
• The ROM, like the RAM, is a random-access memory but the term RAM traditionally
means Random-Access Read/Write Memory.
• Because ROM retains stored data even if power is turned off, they are nonvolatile
memory
